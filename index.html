About

My Works

Android Shopping App

November 2019 - December 2019

At the end of my second-year fall semester at UTSC, one of my courses had us develop an Android app for our culminating project. This was the biggest project of my university years so far and was also the first time I ever worked in a scrum team. Using Android Studio and all the Java skills and design patterns taught in class, my team and I managed to create a working shop app that allowed customers to buy items, employees to restock items, and admins to view activity history. Those are just some of the many features we implemented in this app. Overall, this was a gruelling but fruitful experience and I'm sure the skills learned from this would be applied soon down my career.

Mood

November 2019

I attended the Hack Western 6 hackathon in winter 2019. There, I and a team of four people (myself included), worked on a web app that was able to detect one's current mood, and would generate an endless list of movies and gifs, and select one song, that would help this person with how they're feeling. We made this using various APIs for getting the desired movies, gifs, and songs. Javascript and nodejs for backend purposes. Bootstrap, HTML, and CSS for making the frontend. And we built a custom API that allows us to communicate between backend and frontend. This was by far, the biggest project I've worked on at a hackathon. It proves to me that with a clear end goal and strong dedication, things can be done in a limited amount of time.

Heir to the Throne

March 2017

Another early project was Heir to the Throne, the first game I created using JavaScript, HTML, and CSS. I remember starting as early as I could the moment my high school computer science teacher announced this assignment. I was a huge video game nerd and always dreamed of one day developing my own game or working for a large game company.

This game is a point-and-click adventure game where the player can explore many locations on a map by clicking on its image. The purpose of this game is to find out the traitor among the King's children and arrest them before they either kill the player or the true heir to the throne. In certain areas, there are clues that pertain to the traitor's or the heir's identity and the player can cross-reference those clues to each of the King's children's profiles. The player wins if they arrest the traitor and loses if they arrest the true heir or gets killed by the traitor
